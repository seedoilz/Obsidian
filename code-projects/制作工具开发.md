## 双检锁

```java
private volatile static SingleInstance instance;

public static SingleInstance getInstance(){
	if (instance == null) {
		synchronized (SingleInstance.class) {
			if (instance == null) {
				instance = initInstance();
			}
		}
	}
	return instance;
}
```
