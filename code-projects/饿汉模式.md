### 优点
实现更简单，类一加载就实例化对象。

### 如何实现：
- 将meta属性声明为private static final，并在声明时进行初始化
- 将实例初始化逻辑提取到私有方法initMeta中，保持代码的清晰和可读性

```java
private static final SingleInstance instance = initInstance();
```

与[[双检锁|懒汉模式]]（双检锁）进行对比。
